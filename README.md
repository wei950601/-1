# 家教 Flask App（原型）

## 快速開始
```bash
python -m venv .venv && source .venv/bin/activate  # Windows 用 .venv\Scripts\activate
pip install -r requirements.txt
python app.py  # 或 flask --app app run
# 首次可初始化資料： flask --app app initdb
```
打開瀏覽器前往 http://127.0.0.1:5000

- 首頁：五個彩色入口，樣式依照提供示意圖配色。
- 進入頁面後會以 WebAudio 生成隨機輕音樂，可於「設定」調整左右音量、通知開關、黑白/彩色主題、是否自動打卡。
- 搜尋列：輸入「行事曆 / 打卡 / 問題 / 聯絡簿 / 成績紀錄 / 頭像」等關鍵字會出現結果，點擊自動跳轉。
- 行事曆：左上角「＋」可新增行程（標題、開始/結束時間、兩次提醒選擇），每個行程可刪除。
- 打卡：月曆格上可勾選，若設定中開啟「自動打卡」，當天會自動勾選一次。
- 問題：新增問題、對既有問題填寫/更新答案。
- 聯絡簿：可選日期，以條列方式新增/刪除/儲存當日內容。
- 成績紀錄：先新增科目，再依日期新增分數與排名，並可刪除單筆紀錄。
- 頭像與姓名：可設定顯示在右上角的頭像網址與名稱。
